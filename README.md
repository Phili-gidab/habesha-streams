# Habesha Streams - Marketing Website

Premium East African streaming platform marketing site built with React + Vite.

## 🎬 About

Habesha Streams is the premier streaming platform for Ethiopian, Eritrean & East African cinema, music, and culture. This marketing website drives Android & iOS app downloads.

## 🚀 Features

- **Responsive Design** - Optimized for all devices
- **Fast Performance** - Built with Vite for lightning-fast loading
- **Premium UI** - Cinematic design with gold accents
- **SEO Optimized** - Meta tags and Open Graph support
- **Vercel Ready** - Optimized for Vercel deployment

## 🛠 Tech Stack

- **React 18** - Modern React with hooks
- **Vite** - Fast build tool and dev server
- **CSS3** - Custom design system with animations
- **Google Fonts** - Cinzel (headings) + Inter (body)

## 📱 Sections

1. **Hero** - Cinematic banner with app download CTAs
2. **Content Showcase** - Scrolling thumbnails of available content
3. **Features** - 6-grid showcase of platform capabilities
4. **Pricing** - Simple $5/month subscription plan
5. **Social Proof** - Stats and testimonials
6. **Download CTA** - Final conversion section
7. **Footer** - Links and company information

## 🎨 Brand Colors

- **Primary Gold**: `#C6A75E`
- **Background**: `#0a0804` (deep cinematic dark)
- **Brown Accent**: `#3E2723`
- **Red Accent**: `hsl(0, 65%, 38%)`
- **Emerald**: `hsl(155, 55%, 22%)`

## 🚀 Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📦 Deployment

This site is optimized for Vercel deployment:

1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect the Vite configuration
3. Build command: `npm run build`
4. Output directory: `dist`
